Underlay, AI for Design Sketching

Jun 2023 - Jul 2023

Starting a digital sketch can be challenging, especially when establishing perspective, proportions, and lighting. Underlay simplifies this by generating geometry based on design requirements. Use the tool to type in the object you're designing and it pulls average dimensions, creating a bounding box in the viewport. Modify geometry, lighting, and perspective before downloading a PNG volume. Try it here.

Skills: AI, Google Cloud Platform, Three.js, CSS, HTML, JavaScript, Java
